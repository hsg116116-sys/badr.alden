اريدك تغير الموقع كامل من هاد الموقع تبع الملحمة الى واحد ثاني الى محمصة وهو بدر الدين وهي التفاصيل كاملة : 

🎨 أولاً: الهوية البصرية وتجربة المستخدم (UI/UX & Branding)

1.  الألوان والهوية (Color Palette):

      - الاعتماد على الألوان المستوحاة من شعار "بدر الدين" المرفق:
          - اللون الأساسي (Primary): درجات البني الدافئ للشوكولاتة والقهوة
            (مثال: #5F2E1B أو #4A2312).
          - اللون الثانوي/التمييزي (Accent): البرتقالي الحيوي للمكسرات والتحميص
            (مثال: #E8833B أو #F28E2B).
          - الخلفيات (Backgrounds): تدرجات من الأبيض الدافئ والبيج الخفيف جداً
            لمنح المتجر طابعاً عصرياً ومريحاً للعين.

2.  تحويل ميزات المتجر القديم (المراح) إلى المتجر الجديد (بدر الدين): قم
    باستبدال البطاقات الأربعة المميزة في الواجهة الرئيسية لمتجر اللحوم
    لتصبح متوافقة مع المحمصة كالتالي:

      - البطاقة الأولى (الجودة): من "جودة مضمونة - لحوم طازجة يومياً" ⬅️ إلى
        "جودة مضمونة - مكسرات وبن محمص طازج يومياً". (أيقونة: ساعة أو جودة).
      - البطاقة الثانية (التوصيل): من "توصيل بارد - سيارات مجهزة بالكامل" ⬅️ إلى
        "شحن سريع وآمن - تغليف محكم مفرغ من الهواء للحفاظ على القرمشة والنكهة".
        (أيقونة: شاحنة شحن ذكية).
      - البطاقة الثالثة (الفرز والتحميص): من "ذبح إسلامي - تحت إشراف طبي" ⬅️ إلى
        "فرز يدوي ونقي - منتجات منتقاة بعناية خالية من الشوائب ومحمصة بحرفية".
        (أيقونة: درع حماية أو ورقة شجر خضراء).
      - البطاقة الرابعة (تخصيص الطلب): من "تقطيع احترافي - حسب اختيارك المفضل"
        ⬅️ إلى "طحن وتحميص مخصص - اختر درجة التحميص للمكسرات ونوع الطحن
        للبن". (أيقونة: مطحنة أو ميزان).

🛠️ ثالثاً: هيكلية الأنظمة والواجهات المطلوبة (Core Sub-systems)

يجب بناء المتجر كمنصة متكاملة تحتوي على 3 أنظمة متصلة عبر واجهات برمجية واضحة:

1. تطبيق العميل (Client Front-End):

  - الصفحة الرئيسية (Home Page): هيدر متطور مع شريط بحث ذكي، البانر الإعلاني
    المتحرك، ميزات المحمصة الأربعة، الأقسام الرئيسية، المنتجات الأكثر
    مبيعاً، والفوتر المحدث.
  - صفحة المنتجات (Product Catalog): فلترة ذكية حسب القسم، الوزن، السعر، وخاصية
    البحث الفوري.
  - صفحة تفاصيل المنتج (Product Details): إمكانية اختيار خيارات مخصصة مثل:
      - درجة الطحن للقهوة: (حصى، ناعم جداً/تركي، وسط/فلتر، خشن/فرنسي).
      - درجة التحميص للمكسرات: (نيء، محمص خفيف، محمص وسط، محمص غامق/مملح).
      - الوزن المطلوب: (ثمن كيلو 125غ، ربع كيلو 250غ، نصف كيلو 500غ، كيلو كامل).
  - سلة المشتريات (Shopping Cart) وصفحة الدفع (Checkout): حساب تلقائي للمجموع
    الإجمالي شامل ضريبة القيمة المضافة، إدخال كوبونات الخصم، وتحديد خيارات
    الشحن والدفع (نقداً، فيزا، مدى).

2. لوحة تحكم الإدارة (Admin Dashboard):

  - إدارة المخزون (Inventory Management): إضافة وتحديث منتجات الأقسام الـ 15،
    تحديد كميات المخزن بالكيلوغرام أو بالقطعة، وتلقي تنبيهات عند قرب نفاد
    كمية أي منتج.
  - إدارة الطلبات (Orders Dispatching): تتبع حالة الطلب (قيد التحضير، جاري الطحن
    والتحميص، جاهز للتسليم، مع السائق، تم التوصيل).
  - إدارة السائقين (Drivers Management): تعيين الطلبات للسائقين المتاحين ومراقبة
    أدائهم.

3. لوحة تحكم السائقين والمناديب (Delivery Driver System):

  - واجهة متجاوبة بالكامل للهواتف الذكية تتيح للسائق:
      - استعراض الطلبات المسندة إليه.
      - عرض موقع العميل عبر الخريطة وتفاصيل الاتصال به.
      - تحديث حالة الطلب فور التسليم مع إمكانية تحصيل المبالغ النقدية وتأكيدها.

💻 رابعاً: المعايير البرمجية والتقنية (Technical Stack & Requirements)

يرجى كتابة كود نظيف وقابل للتطوير يتبع المعايير التالية:

1.  TypeScript Static Typing: كتابة واجهات صلبة (Interfaces) لكل من: Product
    ،Category ،Order ،User ،Driver ،CartItem.
2.  State Management: استخدام React Context API أو Zustand لإدارة حالة السلة
    والمستخدم بشكل فعال.
3.  Tailwind CSS: تصميم عصري متجاوب بالكامل (Mobile-First) ومتوافق مع اللغتين
    العربية والانجليزية (RTL / LTR support).
4.  Clean Code Structure: تقسيم المشروع إلى مكونات صغيرة قابلة لإعادة الاستخدام
    (Reusable Components) مثل: Button ،Input ،ProductCard ،Dropdown ،Modal.

⚡ خامساً: أمثلة لهيكل البيانات البرمجي (TypeScript Interfaces)

لمساعدتك في صياغة الكود، استخدم هذا النموذج لتعريف البيانات الأساسية:

export type RoastingLevel = 'raw' | 'light' | 'medium' | 'dark';
export type GrindSize = 'none' | 'turkish' | 'espresso' | 'filter' | 'french_press';

export interface Product {
  id: string;
  name: string;
  arabicName: string;
  category: string; // واحدة من الأقسام الـ 15
  basePricePerKg: number; // السعر الأساسي للكيلو
  unitType: 'weight' | 'piece'; // هل يباع بالوزن أم بالقطعة
  images: string[];
  description: string;
  stockQuantity: number;
  roastingOptionsAvailable: boolean; // هل يدعم تخصيص التحميص
  grindingOptionsAvailable: boolean; // هل يدعم تخصيص الطحن
}

export interface CartItem {
  product: Product;
  selectedWeight: number; // بالغرام (مثال: 250)
  selectedRoast?: RoastingLevel;
  selectedGrind?: GrindSize;
  quantity: number;
  totalPrice: number;
}

export interface Order {
  id: string;
  customerId: string;
  items: CartItem[];
  status: 'pending' | 'preparing' | 'with_driver' | 'delivered' | 'cancelled';
  driverId?: string;
  paymentMethod: 'cash' | 'card' | 'mada';
  deliveryAddress: string;
  totalAmount: number;
  createdAt: Date;
